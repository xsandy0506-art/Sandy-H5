Run the React app:

1. Open terminal and cd into the react-app folder:

```bash
cd C:\Users\30128\Documents\hakka-lantern-h5\react-app
```

2. Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

3. Open the app at the address shown by Vite (default http://localhost:3000).

Notes:
- Put your local images (welcome-gate.jpg, center-lantern.jpg, waterfront.jpg, gallery.jpg, promenade.jpg, day-aerial.jpg, night-aerial.jpg) into the project root `C:\Users\30128\Documents\hakka-lantern-h5` so their paths like `/welcome-gate.jpg` resolve.
- If you prefer the images in `react-app/public/`, move them there and adjust `LOCAL` mapping accordingly.
